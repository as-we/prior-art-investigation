# Prior Art Investigation Agent

Quickly identify if your feature concept already exists, what it's called, and what OSS solutions are available — **before** you code.

## What This Agent Does

1. **Names Your Concept** — Discovers the real technical term (e.g., "smooth seeking" → "time-stretching")
2. **Maps Risk** — Identifies what could go wrong (architectural, performance, adoption)
3. **Lists OSS Options** — Shows existing solutions you could adapt or build on

## When to Use

### Phase 1: Requirements (Quick Check)
- ✅ Have a problem statement?
- ✅ Before moving to design?
- **Use**: `@prior-art-investigation minimal` → 5-10 min, ~150 tokens

### Phase 2: Design (Deep Investigation)
- ✅ Architecture decisions needed?
- ✅ New subsystem design?
- **Use**: `@prior-art-investigation full` → 20-40 min, ~500 tokens

### Phase 3: Unsure Which
- ✅ First time here?
- **Use**: `@prior-art-investigation selector` → 1-2 min (auto-routes)

## Quick Examples

### Example 1: Requirements Phase
```
User: "@prior-art-investigation minimal I need a way to let users search grooves by rhythm pattern"

Agent: 
- Real concept: "Sequence Alignment" or "Rhythm Fingerprinting"
- Q1 First Principles: Is the problem searching grooves or *finding similar rhythms*?
- Q6 Risks: 
  • Assumption: Users understand rhythm patterns
  • Risk: Learning curve blocks adoption
  • Verify: User research on pattern recognition
- Recommend FULL investigation for Design phase
```

### Example 2: Design Phase
```
User: "@prior-art-investigation full We're designing a new ML-based boundary detection system"

Agent:
- Q1-Q7 complete investigation
- Existing OSS: librosa, essentia, madmom
- Architecture options: Real-time vs batch
- Performance benchmarks from prior art
- Recommend: librosa baseline first, then madmom for advanced

Provides full research template saved to .kiro/specs/[feature]/docs/research.md
```

## Commands

| Command | Effect | Time | Tokens |
|---------|--------|------|--------|
| `@prior-art-investigation select` | Auto-detect phase, run appropriate prompt | 1-2 min | ~100 |
| `@prior-art-investigation minimal` | Q1 + Q6 only (Requirements) | 5-10 min | ~150 |
| `@prior-art-investigation full` | Q1-Q7 complete (Design) | 20-40 min | ~500 |

## Outputs

All investigations produce:
- ✅ Concept name confirmed + alternative names
- ✅ OSS options ranked
- ✅ Risk map (assumptions → potential failures)
- ✅ Research template (save to `.kiro/specs/` for Kiro SDD workflows)
